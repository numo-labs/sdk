SDK Sandbox

How to use

  1. Open dev tools console
  2. Enter a query in the console
  3. Hit enter

Example query

  SDK().query({ entities: ['helloworld'] }).on('result', console.log).on('end', () => console.log('done'));

  SDK().query({ entities: ['helloworld'], count: 5 }).on('result', console.log).on('end', () => console.log('done'));
