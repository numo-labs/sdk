Autocomplete

Implements an autocomplete feature to search for tags using a given string.

sdk.autocomplete(<reference>).for(<searchstring>)

keyword type description
reference string The entity you want to autocomplete. E.g. destination
searchstring string The partial world you want to autocomplete

Destination

The destination autocomplete allows you to get a list of destinations based on a partial destination name.

You can acquire a reference to the destination autocomplete by passing in destination as the reference string. sdk.autocomplete('destination');

Signature
.for(string: input, string: departureAirport, function: callback)

keyword type description
input string The partial destination name you want to autocomplete
departureAirport string The departure airport where you want to depart from
callback function The callback function to retrieve the results

Example

const autocomplete = sdk.autocomplete('destination');

autocomplete.for('spa', <airport_code>, function (err, result) {});
Departure airport

The departure arport autocomplete allows you to get a list of airports based on a partial airport name.

You can acquire a reference to the departure airport autocomplete by passing in departure_airport as the reference string. sdk.autocomplete('departure_airport');

Signature
.for(string: input, function: callback)

keyword type description
input string The partial airport name you want to autocomplete
callback function The callback function to retrieve the results

Example

const autocomplete = sdk.autocomplete('departure_airport');

autocomplete.for('lon', function (err, result) {});
